# Development Books Price Calculator

TDD Kata for BNP Paribas Fortis selection process.

## Problem

Calculate the price of a basket of software development books with discounts based on book diversity.

## Rules

- Base price: 50 EUR per book
- Discounts on different books:
  - 2 different: 5%
  - 3 different: 10%
  - 4 different: 20%
  - 5 different: 25%

## Build

```bash
mvn clean install
```

## Test

```bash
mvn test
```
